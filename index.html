Setup Guide
Go live in 20 minutes.
Follow every step carefully. Free forever. No credit card.
1
FREE

📥 Download This File

Download ClipBySu-FINAL.html to your computer.

Open it in any text editor (Notepad, VS Code, Notepad++).

You will need to paste your Firebase config inside it — explained in Step 4.

2
FREE

🔥 Create a Firebase Account

Go to firebase.google.com → Click Get Started → Sign in with your Google account.

Firebase is 100% free for small projects like ClipBySu (Spark Plan).

https://firebase.google.com
3
REQUIRED

➕ Create a Firebase Project

In Firebase Console → Click "Add project"

Name it: clipbysu → Disable Google Analytics (not needed) → Click Create project

Wait ~30 seconds for it to be created → Click Continue

4
REQUIRED

🌐 Register a Web App

Inside your project → Click the Web icon (</>) to add a web app

App nickname: ClipBySu → Click Register app

You will see a firebaseConfig object. Copy it — it looks like this:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "clipbysu.firebaseapp.com", databaseURL: "https://clipbysu-default-rtdb.firebaseio.com", projectId: "clipbysu", storageBucket: "clipbysu.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };
⚠️ Copy this EXACT config. You need all 7 values. Replace the placeholder values in the HTML file with yours.
5
REQUIRED

🗄️ Enable Realtime Database

In Firebase Console left menu → Click Build → Click Realtime Database

Click "Create Database" → Choose location: United States (us-central1) → Click Next

Select "Start in test mode" → Click Enable

⚠️ Test mode allows read/write for 30 days. After 30 days update rules to keep it working — see Step 9.
6
REQUIRED

✏️ Paste Config Into HTML File

Open your downloaded HTML file in a text editor.

Find this section near the top (around line 20):

const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", databaseURL: "https://YOUR_PROJECT_ID-default-rtdb...", projectId: "YOUR_PROJECT_ID", ... };

Replace all YOUR_... values with your actual Firebase config values from Step 4.

Save the file as index.html

7
FREE

🐙 Upload to GitHub Pages

Go to github.com → Sign up / Log in → Click "+" → New repository

Name: clipbysu · Set to Public · Check ✅ Add README → Create repository

Click Add file → Upload files → Drop your index.html → Commit changes

Go to Settings → Pages → Branch: main → / (root) → Save

Wait 2 minutes. Your site is live at:

https://YOUR-USERNAME.github.io/clipbysu
8
REQUIRED

🔓 Add GitHub Domain to Firebase

Firebase blocks requests from unknown domains. You must whitelist your GitHub Pages URL.

In Firebase Console → Authentication → Settings → Authorized domains

Click Add domain → Enter your GitHub Pages URL:

YOUR-USERNAME.github.io

Click Add. Now your site can talk to Firebase.

9
REQUIRED after 30 days

🛡️ Update Firebase Security Rules

Firebase test mode expires after 30 days. To keep it working, update the rules.

In Firebase Console → Realtime Database → Rules → Replace with:

{ "rules": { ".read": true, ".write": true } }

Click Publish. This keeps ClipBySu working forever.

💡 For better security later, you can add Firebase Authentication and restrict access to only your account.
